Some information about this site for the nerds.

Why build from scratch?

I wanted to go on a spiritual journey where I learn everything from scratch. I wanted to build the following:

  • Integrating cloud services
  • Developing web services
  • Basic web development
  • User interface design

Architecture

This website is hosted on Amazon Web Services consisting of the following components:

  • S3 - Serves static web pages
  • API Gateway - Front door for API calls to other services hosted on AWS
  • Lambda - Serverless code for web services
  • DynamoDB - Store data from API calls
